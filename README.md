# THPSBot v4.1

## What is this!?
THPSBot is a bot for the [THPS Speedrun Discord](https://thps.run/discord). It is a bot specially designed by Anastasia that combines multiple functions.

## Note
This bot is mostly plug-and-play. However, if you use this for your project or fork it, please know that `thpsrun.py` will not work.

If you want to assist in this project's development for the THPS Speedrun Discord, let Anastasia know and he can make you an API key.
